## Exercicio 2
```shell
João é um comerciante e precisa automatizar o seu processo de vendas
Para automatizar este processo João chamou a empresa SUA_EMPRESA(Você aluno)
O problema de joão é que ele precisa cadastrar os seus produtos
e vender estes produtos para seus clientes.

Baseado nesta demanda, faça um programa que ajude o João a vender mais.

Este programa precisa ter uma entrada e só sair do programa, quando o João
quiser.

E ter opcão de:
- Cadastrar produtos
- Vender produtos (Criar pedidos, com clientes)
- Mostrar relatório dos mesmos
```
